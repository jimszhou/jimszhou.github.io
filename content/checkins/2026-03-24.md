## Core Technique: 2D Array Traversal Tricks

Two essential matrix manipulation patterns: **rotation via decomposition** and **spiral traversal via boundary shrinking**. These aren't algorithmic frameworks — they're tricks that are easy once you've seen them, but hard to invent on the spot.

**What it is:** Techniques for in-place matrix rotation (clockwise/counterclockwise) and spiral-order traversal of 2D arrays.

**When to use it:** Matrix transformation problems, spiral printing/generation, or any problem requiring non-standard 2D traversal order.

**Why use it:** Avoids complex coordinate mapping. Rotation decomposes into two simple operations (transpose + reverse). Spiral traversal uses four shrinking boundaries instead of tracking visited cells.

```python
# Clockwise 90° rotation = transpose + reverse each row
def rotate_cw(matrix):
    n = len(matrix)
    for i in range(n):
        for j in range(i, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    for row in matrix:
        row.reverse()

# Spiral traversal template
def spiral(matrix):
    m, n = len(matrix), len(matrix[0])
    top, bottom, left, right = 0, m - 1, 0, n - 1
    res = []
    while len(res) < m * n:
        for j in range(left, right + 1): res.append(matrix[top][j])
        top += 1
        for i in range(top, bottom + 1): res.append(matrix[i][right])
        right -= 1
        if top <= bottom:
            for j in range(right, left - 1, -1): res.append(matrix[bottom][j])
            bottom -= 1
        if left <= right:
            for i in range(bottom, top - 1, -1): res.append(matrix[i][left])
            left += 1
    return res
```

## Problems Studied

### 48. Rotate Image

Rotate an n x n matrix 90 degrees clockwise in-place. The key insight: don't try to map coordinates directly. Instead, **transpose along the main diagonal, then reverse each row**. Two simple passes replace one complex one. For counterclockwise rotation, transpose along the anti-diagonal instead.

```python
def rotate(matrix):
    n = len(matrix)
    for i in range(n):
        for j in range(i, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    for row in matrix:
        row.reverse()
```

### 54. Spiral Matrix

Return all elements of an m x n matrix in spiral order. Maintain four boundaries (`top`, `bottom`, `left`, `right`) and traverse right, down, left, up — shrinking the corresponding boundary after each pass. The boundary checks before the left and up passes prevent double-counting in non-square matrices.

```python
def spiralOrder(matrix):
    m, n = len(matrix), len(matrix[0])
    top, bottom, left, right = 0, m - 1, 0, n - 1
    res = []
    while len(res) < m * n:
        if top <= bottom:
            for j in range(left, right + 1):
                res.append(matrix[top][j])
            top += 1
        if left <= right:
            for i in range(top, bottom + 1):
                res.append(matrix[i][right])
            right -= 1
        if top <= bottom:
            for j in range(right, left - 1, -1):
                res.append(matrix[bottom][j])
            bottom -= 1
        if left <= right:
            for i in range(bottom, top - 1, -1):
                res.append(matrix[i][left])
            left += 1
    return res
```

### 59. Spiral Matrix II

Generate an n x n matrix filled with 1 to n² in spiral order. Same boundary-shrinking approach as 54, but write incrementing numbers instead of reading.

```python
def generateMatrix(n):
    matrix = [[0] * n for _ in range(n)]
    top, bottom, left, right = 0, n - 1, 0, n - 1
    num = 1
    while num <= n * n:
        for j in range(left, right + 1): matrix[top][j] = num; num += 1
        top += 1
        for i in range(top, bottom + 1): matrix[i][right] = num; num += 1
        right -= 1
        if top <= bottom:
            for j in range(right, left - 1, -1): matrix[bottom][j] = num; num += 1
            bottom -= 1
        if left <= right:
            for i in range(bottom, top - 1, -1): matrix[i][left] = num; num += 1
            left += 1
    return matrix
```

## Key Patterns to Remember

| Pattern | When to Use | Examples |
|---|---|---|
| Transpose + reverse rows | Clockwise 90 degree rotation | 48 |
| Anti-diagonal transpose + reverse rows | Counterclockwise 90 degree rotation | 48 (variant) |
| Reverse whole then reverse parts | In-place order rearrangement | 151, 61 |
| Four-boundary spiral | Spiral traversal or generation | 54, 59 |

## Rotation Decomposition

| Rotation | Step 1 | Step 2 |
|---|---|---|
| **CW 90** | Transpose (main diagonal) | Reverse each row |
| **CCW 90** | Transpose (anti-diagonal) | Reverse each row |
| **180** | Reverse each row | Reverse each column |
