## Core Technique: Difference Array

A difference array is the inverse of a prefix sum. Instead of querying range sums efficiently, it lets you **apply range updates efficiently**. Given an array, build a diff array where `diff[i] = nums[i] - nums[i-1]`. To add a value `val` to every element in `nums[l..r]`, you only need two operations:

```python
diff[l] += val
diff[r + 1] -= val
```

After all updates, reconstruct the final array by taking the prefix sum of `diff`. This turns O(n) per update into O(1) per update, with a single O(n) reconstruction pass at the end.

**What it is:** A technique that records the *change* at boundaries rather than modifying every element in a range. The difference array `diff[i]` represents how much `nums[i]` differs from `nums[i-1]`.

**When to use it:** When you have multiple range update operations (add/subtract a value to all elements in `[l, r]`) and only need the final result after all updates are applied. It's the go-to pattern for batch interval modifications.

**Why use it:** Naively applying `k` range updates on an array of size `n` costs O(k × n). With a difference array, each update is O(1) and the final reconstruction is O(n), making the total O(k + n).

```python
# Difference array template
def apply_range_updates(length, updates):
    diff = [0] * (length + 1)  # extra slot to avoid bounds checking

    for l, r, val in updates:
        diff[l] += val
        diff[r + 1] -= val

    # Reconstruct final array via prefix sum
    result = [0] * length
    result[0] = diff[0]
    for i in range(1, length):
        result[i] = result[i - 1] + diff[i]

    return result
```

## Problems Studied

### 370. Range Addition 🔒

Given a `length` and a list of operations `[startIndex, endIndex, inc]`, apply each increment to the range and return the final array. This is the textbook difference array problem — create a diff array, mark `+inc` at `start` and `-inc` at `end + 1`, then take prefix sum to get the answer.

```python
def getModifiedArray(length, updates):
    diff = [0] * (length + 1)
    for start, end, inc in updates:
        diff[start] += inc
        diff[end + 1] -= inc

    result = [0] * length
    result[0] = diff[0]
    for i in range(1, length):
        result[i] = result[i - 1] + diff[i]
    return result
```

### 1109. Corporate Flight Bookings

There are `n` flights labeled `1` to `n`. Given bookings `[first, last, seats]`, return an array where `answer[i]` is the total seats reserved for flight `i`. Same idea as 370 but with 1-indexed flights. Apply `+seats` at `first - 1` and `-seats` at `last` in the diff array, then prefix sum.

```python
def corpFlightBookings(bookings, n):
    diff = [0] * (n + 1)
    for first, last, seats in bookings:
        diff[first - 1] += seats
        diff[last] -= seats

    result = [0] * n
    result[0] = diff[0]
    for i in range(1, n):
        result[i] = result[i - 1] + diff[i]
    return result
```

### 1094. Car Pooling

Given `trips = [numPassengers, from, to]` and a car with `capacity`, determine if it's possible to pick up and drop off all passengers without exceeding capacity. The key insight: passengers board at `from` and leave at `to`, so treat each trip as a range update of `+numPassengers` on `[from, to)`. Note `to` is exclusive — passengers are gone by that stop. Build the diff array, prefix sum it, and check if any stop exceeds capacity.

```python
def carPooling(trips, capacity):
    diff = [0] * 1001  # stops range 0..1000
    for num, start, end in trips:
        diff[start] += num
        diff[end] -= num  # end is exclusive, so no +1

    current = 0
    for d in diff:
        current += d
        if current > capacity:
            return False
    return True
```

## Key Patterns to Remember

| Pattern | When to Use | Examples |
|---|---|---|
| Basic difference array | Batch range additions, return final state | 370, 1109 |
| Diff array + capacity check | Range updates then validate a constraint | 1094 |
| Difference array + prefix sum | They are inverses — diff array undoes prefix sum and vice versa | all |
| 1-indexed adjustment | Problem uses 1-based indexing | 1109 |
| Exclusive end range | Interval is `[start, end)` not `[start, end]` | 1094 |

## Difference Array vs Prefix Sum

| | Prefix Sum | Difference Array |
|---|---|---|
| **Optimizes** | Range queries | Range updates |
| **Build** | `prefix[i] = prefix[i-1] + nums[i]` | `diff[i] = nums[i] - nums[i-1]` |
| **Reconstruct** | Subtract to get original | Prefix sum to get original |
| **Relationship** | Inverse of each other | Inverse of each other |
