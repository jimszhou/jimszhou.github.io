## Core Technique: Sliding Window

A sliding window is a fast/slow two-pointer pattern where `[left, right)` defines a window that only expands or shrinks — neither pointer ever backtracks. This means every element enters and leaves the window at most once, giving **O(N)** time versus O(N^2) brute force.

**When to use:** Finding the shortest/longest substring or subarray satisfying some condition (character frequency match, uniqueness, sum constraint).

**Template:**

```python
from collections import defaultdict

def sliding_window(s: str, t: str):
    need = defaultdict(int)
    window = defaultdict(int)
    for c in t:
        need[c] += 1
    left = right = 0
    valid = 0  # number of characters whose frequency requirement is met

    while right < len(s):
        c = s[right]
        right += 1
        # expand: update window data
        if c in need:
            window[c] += 1
            if window[c] == need[c]:
                valid += 1

        # shrink condition varies per problem
        while valid == len(need):  # or other condition
            d = s[left]
            left += 1
            # contract: mirror the expand logic
            if d in need:
                if window[d] == need[d]:
                    valid -= 1
                window[d] -= 1
```

## Problems Studied

### 76. Minimum Window Substring

Find the shortest substring of `s` containing all characters of `t`. Expand until all characters are covered, then shrink from the left to minimize length. Update the answer during the shrink phase.

```python
def minWindow(s, t):
    from collections import defaultdict
    need, window = defaultdict(int), defaultdict(int)
    for c in t:
        need[c] += 1
    left = right = valid = 0
    start, length = 0, float('inf')
    while right < len(s):
        c = s[right]
        right += 1
        if c in need:
            window[c] += 1
            if window[c] == need[c]:
                valid += 1
        while valid == len(need):
            if right - left < length:
                start, length = left, right - left
            d = s[left]
            left += 1
            if d in need:
                if window[d] == need[d]:
                    valid -= 1
                window[d] -= 1
    return "" if length == float('inf') else s[start:start + length]
```

### 567. Permutation in String

Check if any permutation of `s1` exists as a substring in `s2`. Fixed-size window of length `len(s1)` — when `valid == len(need)`, return True.

```python
def checkInclusion(s1, s2):
    from collections import defaultdict
    need, window = defaultdict(int), defaultdict(int)
    for c in s1:
        need[c] += 1
    left = right = valid = 0
    while right < len(s2):
        c = s2[right]
        right += 1
        if c in need:
            window[c] += 1
            if window[c] == need[c]:
                valid += 1
        while right - left >= len(s1):
            if valid == len(need):
                return True
            d = s2[left]
            left += 1
            if d in need:
                if window[d] == need[d]:
                    valid -= 1
                window[d] -= 1
    return False
```

### 438. Find All Anagrams in a String

Same as 567 but collect all starting indices instead of returning on first match.

```python
def findAnagrams(s, p):
    from collections import defaultdict
    need, window = defaultdict(int), defaultdict(int)
    for c in p:
        need[c] += 1
    left = right = valid = 0
    res = []
    while right < len(s):
        c = s[right]
        right += 1
        if c in need:
            window[c] += 1
            if window[c] == need[c]:
                valid += 1
        while right - left >= len(p):
            if valid == len(need):
                res.append(left)
            d = s[left]
            left += 1
            if d in need:
                if window[d] == need[d]:
                    valid -= 1
                window[d] -= 1
    return res
```

### 3. Longest Substring Without Repeating Characters

Expand the window; when a duplicate enters, shrink until the duplicate is removed. Track the max window size throughout.

```python
def lengthOfLongestSubstring(s):
    window = {}
    left = right = res = 0
    while right < len(s):
        c = s[right]
        right += 1
        window[c] = window.get(c, 0) + 1
        while window[c] > 1:
            d = s[left]
            left += 1
            window[d] -= 1
        res = max(res, right - left)
    return res
```

## Key Patterns to Remember

| Pattern | Shrink Condition | Update Result | Problems |
|---|---|---|---|
| Minimum window covering target | All target chars satisfied | During shrink (minimize) | 76 |
| Fixed-length permutation check | Window size >= target length | During shrink (exact match) | 567, 438 |
| Longest without duplicates | Duplicate char in window | After shrink (maximize) | 3 |
